# Opportunities Database Africa

A comprehensive platform connecting African students and professionals with educational and career opportunities across the continent and beyond.

## 🌟 Features

- **Scholarships**: Browse and apply for scholarships worldwide
- **Internships**: Find internship opportunities across various industries
- **Exchange Programs**: Explore international exchange opportunities
- **Research Grants**: Access funding for research projects
- **User-Friendly Interface**: Modern, responsive design for all devices
- **Advanced Search**: Filter opportunities by location, duration, and type
- **Real-time Updates**: Stay informed about new opportunities

## 🚀 Tech Stack

- **Frontend**: Next.js 15.2.3, React
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Type Safety**: TypeScript
- **Development**: Node.js

## 📋 Prerequisites

Before running this project, make sure you have:

- Node.js (v18 or higher)
- npm (v9 or higher)
- Git

## 🛠️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/obure1always/opportunities-database-africa.git
   ```

2. Navigate to the project directory:
   ```bash
   cd opportunities-database-africa
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser
